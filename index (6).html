Step-by-Step: Connect Your First Workspace

1
Go to spanveil.com and click Sign In
Click "Sign In" in the top navigation. You'll be taken to the demo dashboard immediately. To connect your real data, click "Connect API key" in the blue bar at the top.
2
Get your OpenAI API key
Log into platform.openai.com → click your name → "API Keys" → "Create new secret key". Copy the key starting with sk-
3
Paste your key into Spanveil
Paste the key into the "OpenAI API Key" field. Your key is used read-only and never stored on our servers.
4
Set your monthly budget limit
Enter your monthly AI spend budget. Spanveil alerts you at 80% and auto-pauses agents at 100% if enabled.
5
Your agents appear instantly
All agents in your workspace appear on the dashboard with real-time cost data, status, and usage breakdown.

How-To Guides

💰
How to reduce agent costs by 40%
The most common waste is redundant tool calls. In the Audit Log, look for agents with over 500 calls per day. Reducing tool calls by half cuts costs by 60%. Use the Guardrails panel to set per-agent daily caps.
🔁
How to detect and stop a runaway loop
Go to Guardrails → enable "Loop detection & circuit breaker". Set threshold to 100 calls per 60 seconds. Any agent exceeding this auto-pauses with an immediate alert sent to you.
📊
How to generate a compliance report
Go to Audit Log → click "Export CSV". The export includes agent name, timestamp, action type, token usage, cost, and outcome — everything needed for EU AI Act compliance documentation.
🌐
How to add multiple model providers
Click "Add Workspace" in the nav. Each model provider gets its own workspace. Costs are unified in the main dashboard view across all providers.

Frequently Asked Questions

AI Agent Glossary

TermDefinition
AI AgentA software system powered by an LLM that can autonomously take actions, call tools, and complete multi-step tasks without constant human input.
TokenThe basic unit of text processed by an LLM. Roughly 4 characters. All LLM costs are calculated in tokens — input plus output.
Tool CallWhen an agent calls an external function or API — like searching the web, querying a database, or sending an email. Often the biggest cost driver.
Runaway LoopWhen an agent gets stuck repeating the same actions indefinitely. Can burn thousands of dollars in minutes without detection.
Agent SprawlThe uncontrolled proliferation of AI agents across an enterprise with no central visibility or governance.
GuardrailAn automated rule that limits or stops agent behavior — like a budget cap that auto-pauses an agent when it exceeds a limit.
Audit LogAn immutable, timestamped record of every action an agent takes. Required by EU AI Act and SOC 2 compliance frameworks.
Agent FinOpsFinancial Operations for AI agents — the practice of managing and optimizing agent infrastructure costs. What Spanveil enables.
