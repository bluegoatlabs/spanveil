Step-by-Step: Connect Your First Workspace

1
Go to spanveil.com and click Sign In
Click "Sign In" in the top navigation. You'll be taken to the demo dashboard. To connect real data, click "Connect API key" in the blue bar.
2
Get your OpenAI API key
Log into platform.openai.com → click your name → "API Keys" → "Create new secret key".
3
Paste your key into Spanveil
Your key is used read-only and never stored on our servers.
4
Set your monthly budget limit
Spanveil alerts you at 80% and auto-pauses agents at 100% if enabled.
5
Your agents appear instantly
All agents appear with real-time cost data, status, and usage breakdown.

AI Agent Glossary

TermDefinition
AI AgentA software system powered by an LLM that can autonomously take actions and complete multi-step tasks.
TokenThe basic unit of text processed by an LLM. All LLM costs are calculated in tokens.
Tool CallWhen an agent calls an external function or API. Often the biggest cost driver.
Runaway LoopWhen an agent gets stuck repeating actions indefinitely, burning money fast.
Agent SprawlUncontrolled proliferation of AI agents with no central visibility.
GuardrailAn automated rule that limits or stops agent behavior.
Audit LogAn immutable record of every action an agent takes.
Agent FinOpsFinancial Operations for AI agents — what Spanveil enables.
